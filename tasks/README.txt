Esta carpeta se usará para la comunicación entre el bot de Telegram y el proceso principal de Baileys (index.js).

- El bot de Telegram escribirá archivos de tarea aquí (por ejemplo, pairing-<numero>.json).
- El proceso principal leerá y procesará las tareas, y escribirá los resultados (por ejemplo, result-<numero>.json).

No borres esta carpeta.
